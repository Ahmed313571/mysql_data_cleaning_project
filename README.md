# SQL Data Cleaning Project

## Overview
This project demonstrates data cleaning techniques in MySQL using imported Movies data.

## Skills Used
- ALTER TABLE
- UPDATE
- STR_TO_DATE()
- REPLACE()
- Data type conversion
- DECIMAL formatting
- CONVERT
- CAST

## Tools
- MySQL Workbench
- GitHub

## Key Tasks
- Converted text dates into DATE format
- Removed currency symbols from text fields
- Converted TEXT columns into DECIMAL values
- Repaired Corrupt data
